## 选择你的代码右键打开目录

## 然后手动添加文件夹和文件，即创建文件夹以及文件

## 找到GitHub项目右击打开git shell命令行

## 使用git add dir/files的方式添加文件，注意是 “/” 而不是“\”

## 如果提示失败请用带-f 参数来添加文件

## 添加完成之后使用git commit添加说明
   输入上面的git commit之后会直接打开你操作系统默认的文本编辑器
输入你的更新说明

## 关闭文本编辑器之后会动提交更新说明

## 接着使用git push 来提交我们刚刚的修改
